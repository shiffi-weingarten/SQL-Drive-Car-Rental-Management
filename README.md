# SQL Drive - Car Rental Management System 🚗💨
**מאת: שרי ושפי וינגרטן**

מערכת ניהול חכמה ומקיפה לחברת השכרת רכבים, המבוססת על מסד נתונים יחסי (Relational Database) ב-SQL Server. המערכת מספקת פתרון מקצה לקצה: מניהול ציי רכב וסניפים ועד למערכת חישוב עלויות ובונוסים לעובדים.

## 🏗 ארכיטקטורת נתונים (Database Schema)
[cite_start]המערכת מורכבת מ-8 טבלאות מרכזיות המקושרות ביניהן בקשרי גומלין מורכבים (Foreign Keys)[cite: 34]:
* [cite_start]**Cars & TypeCar:** ניהול מלא של מלאי הרכבים, דגמים וקטגוריות[cite: 9, 34].
* [cite_start]**Rentals & RentalsHistory:** מעקב אחר השכרות פעילות וארכיון היסטורי מלא[cite: 9, 21].
* [cite_start]**Workers & Branches:** ניהול עובדים (כולל מערכת בונוסים) ופריסת סניפים ארצית[cite: 9, 11].
* [cite_start]**Clients:** מאגר לקוחות הכולל בקרת גיל (מינימום 18)[cite: 9].

## ✨ יכולות טכנולוגיות בולטות

### ⚙️ פרוצדורות חכמות (Stored Procedures)
* [cite_start]**`usp_RentCar`**: פרוצדורה המבצעת השכרה: מחשבת מחיר אוטומטית (לפי שעה/יום/שבוע) ומעדכנת בונוס לעובד המבצע[cite: 13, 36].
* **`usp_returncar`**: מנהלת את סיום ההשכרה: העברת הנתונים להיסטוריה, עדכון סטטוס ומחיקה מהטבלה הדינמית[cite: 22, 37].
* **`usp_gettopworker`**: כלי BI לשליפת עובדים מצטיינים לפי סניף וגובה הבונוס המצטבר[cite: 14, 38].

### 🛡 בקרת נתונים ואוטומציה (Triggers & Functions)
* **מניעת כפל השכרות**: טריגר `trg_Rentals_PreventOverlapInsert` (מסוג `INSTEAD OF INSERT`) המונע השכרת רכב תפוס בתאריכים חופפים[cite: 23, 35].
* [cite_start]**פונקציית חישוב**: `ufn_GetRentalDurationInDays` לחישוב מדויק של ימי השכרה בתוך שאילתות ופרוצדורות[cite: 25, 40].
* **אילוצי תקינות**: הגדרת אילוצי `CHECK` לגיל לקוח וערכי `DEFAULT` לבונוס עובד[cite: 9].

### 📊 ניתוח נתונים (Views & BI Queries)
* [cite_start]**`vw_BranchCarDetails`**: תצוגה המשלבת `Joins` מורכבים להצגת תמונה מאוחדת של רכב, סניף ומחיר[cite: 18, 27].
* **שאילתות מתקדמות**: שימוש ב-`Sub-queries` ו-`Group By` לניתוח פריסת רכבים וביצועי כוח אדם[cite: 28, 29, 39].
